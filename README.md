# MVPDEMO
## mvp小demo，主要为工具封装

### 一个mvp简单demo 主要为ZXLib封装工具类

### 工具类zxlib开源使用

#### 跟项目添加<br>
```java
allprojects {
		repositories {<br>
			...
			maven { url 'https://jitpack.io' }<br>
		}<br>
	}
	
```	
#### 项目中添加<br>
```java
dependencies {<br>
	      compile 'com.github.zhangxiang0316.MVPDEMO:zxlib:v1.0.1'<br>
	}
